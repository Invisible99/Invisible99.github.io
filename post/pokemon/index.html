Pokémon - 2D

While working on this project I learned a lot about how Pokémon works and this is my first 2d game that I made on my own with the use of the DAE game engine.

This project made me realise there is a lot going on in Pokémon. Like catch rate, the rate Pokémon’s appear, IV’s, EV’s and so much more.
